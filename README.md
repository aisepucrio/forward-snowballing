# SnowMap: A Tool to Support the Snowballing Process in Literature Reviews

This repository contains the developed tool and the corresponding analyses for forward and backward snowballing, as well as comparisons with other tools existing in the literature.

## Main structure

- `Backward_Analisys/` - Analyses and data for backward snowballing.
- `Forward_Analisys/` - Analyses and data for forward snowballing.
- `Framework_snowballing/` - Scripts of the developed framework.

## Important note

Each main subdirectory contains its own `README.md` with detailed instructions on how to run the scripts and use the data included in it.

> Before running any script or analysis, check the `README.md` inside the corresponding folder.

## Getting started

1. Open the project folder in your development environment.
2. Navigate to the subfolder of interest.
3. Read the `README.md` in that subfolder for specific execution instructions.
